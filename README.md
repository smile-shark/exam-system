# 考试系统文档

## 需求

#### 客户

1. 客户端注册
2. 客户端登录
3. 客户端考试
4. 查看考试记录
5. 查看考试详细信息

#### 管理

1. 管理员添加数据
    1. 导入固定格式的excel表格读取数据0
    2. 手动选择课程小节添加数据
2. 管理员创建试卷
3. 管理员发布考试试卷（选择固定时间段）
4. 查看学生考试成绩/信息
5. 导出学生统计成绩（excel文件，可选择某一次考试）

#### 数据

1. 初始数据部分从api接口获取
2. 管理员可以添加数据
3. 后期添加ai出题
